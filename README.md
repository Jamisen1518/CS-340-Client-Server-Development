# CS 340 Portfolio Reflection – Grazioso Salvare Dashboard Project

## How do you write programs that are maintainable, readable, and adaptable?

I write programs that are maintainable, readable, and adaptable by organizing my code into separate sections and using clear, descriptive names for variables and functions. I also include comments when needed so that the purpose of the code is easy to understand. In Project One, I created a CRUD Python module that handled all communication with the database. By keeping the database functions separate from the dashboard code, it was much easier to manage, test, and troubleshoot the project.

One of the biggest advantages of this approach was that I could reuse the same CRUD module throughout Project Two instead of rewriting database code. This saved time and made the application easier to update. In the future, I could use the same module as the foundation for other projects that require database access, making development more efficient.

## How do you approach a problem as a computer scientist?

When solving a problem, I start by reviewing the requirements and breaking the project into smaller, more manageable tasks. For the Grazioso Salvare dashboard, I first focused on understanding the database and how the data was organized. Once I established a connection to the database through the CRUD module, I built and tested each dashboard component step by step.

This project was different from many previous assignments because it required combining several technologies, including MongoDB, Python, Dash, and data visualization tools. It showed me the importance of planning, testing, and building projects in stages. In the future, I would continue using this method to design databases and applications that meet a client's specific needs while remaining organized and scalable.

## What do computer scientists do, and why does it matter?

Computer scientists develop technology solutions that help people and organizations work more effectively. They design software, manage data, solve technical problems, and create systems that improve efficiency and decision-making.

This project demonstrated how technology can turn large amounts of data into useful information. The dashboard allows users to search, filter, and visualize animal rescue data quickly and easily. For an organization like Grazioso Salvare, this can improve productivity and support better decision-making by providing access to important information in a clear and organized way.
